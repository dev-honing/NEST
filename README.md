# NestJS

## 1. CNA 설치
1-1. nextjs-blog\에 설치
```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/learn-starter"
```

1-2. app\에 설치
```
npx create-next-app@latest app
```

## 2. 디렉토리 이동
2-1. nextjs-blog\로 이동
```
cd nextjs-blog
```
2-2. app\으로 이동
```
cd app
```

## 3. NextJS 실행
```
npm run dev
```
